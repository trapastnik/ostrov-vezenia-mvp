### ~~1. API Почты России — авторизация~~ ✅ Решено (2026-02-19)

Авторизация к API Почты России настроена. Работают:

- Нормализация адреса, ФИО, телефона
- Тест-интерфейс в админке (`/pochta`)

### ~~2. Публичный тариф вместо договорного~~ ✅ Решено (2026-02-19)

Реализовано сравнение тарифов (`compare_tariffs()` в pochta.py). Отображается экономия в карточке заказа.

**Примечание:** savings=0 т.к. нет скидки по договору (#66 в BACKLOG). Для реальной экономии нужен корпоративный договор с Почтой России.
